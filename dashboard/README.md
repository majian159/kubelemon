# KubeLemon

## Environment Prepare

Install `node_modules`:

```bash
yarn
```
```

## Provided Scripts

### Start project

```bash
yarn start
```

### Build project

```bash
yarn build
```

### Check code style

```bash
yarn lint
```

You can also use script to auto fix some lint error:

```bash
yarn lint:fix
```

### Test code

```bash
yarn test
```
